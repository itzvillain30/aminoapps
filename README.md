## Getting Started

### Installation

1. Clone the repository:

   ```sh
   git clone https://github.com/itzvillain30/aminoapps
   cd aminoapps
   ```

2. Execute The Repository In Codespace:

   ```sh
   chmod +x *
   python main.py
   ```

3. Install the required libraries:

   ```sh
   python install.py
   ```

4. Run Command :
   ```sh
   python main.py
   ```
